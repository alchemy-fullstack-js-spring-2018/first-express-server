# How To Use
## GET requests to /teas
* `/teas` 
    * returns **all teas in db**

* `teas/id`
    * returns **tea with that id**

## POST request to /teas
* adds tea to db

## PUT request to /teas/id
* updates **tea with that id**

## DELETE request to /teas/id
* deletes **tea with that id**
